# Namma HomeStay Android Project

A warm, rural-themed Android application for managing HomeStays.

## Features
- **Dashboard**: View all available HomeStays with real-time menu updates.
- **Home Profile**: Manage listings, upload room/farm photos, and track verification.
- **Daily Menu**: Update local food menus with live sync.
- **Inquiry Box**: Receive traveler messages and contact them via Call/WhatsApp.
- **Local Guide**: Highlight nearby waterfalls, trekking spots, and viewpoints.

## Tech Stack
- **Language**: Kotlin
- **Database**: Firebase Firestore
- **UI**: Material Design (XML)
- **Libraries**: Glide, ImagePicker, RecyclerView, CardView

## Setup Instructions
1. Open this folder in **Android Studio**.
2. Go to the Firebase Console and create a new project.
3. Add an Android App with package name `com.nammahomestay.app`.
4. Download `google-services.json` and place it in the `app/` folder.
5. Enable **Firestore Database** in the Firebase Console.
6. Build and Run!
