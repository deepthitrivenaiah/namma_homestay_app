Namma-HomeStay is a "Simplified Host Portal." It is a dedicated app for small rural home-stays. It focuses on the "Local Experience. "Today’s Menu" (e.g., Akki Rotti, Bamboo shoot curry) and their "Daily Rate." It’s designed to be managed by a farmer or a homemaker.

 App Usage & User Flow 
 ● Daily Menu: Update what is special for dinner today.
 ● Inquiry Box: See messages and "Call" buttons from travelers.
